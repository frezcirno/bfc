# Brainf\*ck Language Interpreter and Compiler in OCaml

This is a simple BF language interpreter and compiler written in OCaml. The interpreter reads a BF source code from a file and executes it. The compiler reads a BF source code from a file and compiles it to an executable file.

## Usage

Interpreter:
```bash
$ dune exec bfc interpret ./test/hello.bf
Hello World!
```

Compiler:

$ dune exec bfc compile ./test/mandelbrot.bf
